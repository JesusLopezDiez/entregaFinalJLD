# entregaFinalJLD
Entrega del proyecto de DAM a traves de git


Importante 

Seguir las instrucciones de la guia basica de usuario (Otros documentos anexados) para instalar el programa correctamente.
Es muy recomendable leer el documento del proyecto antes de operar con el, aunque con la guia basica seria suficiente.

El profesor puede ejecutar el jar directamente si desea ahorrarse la instalacion en el equipo

En la instalacion solo se ha incluido el propio programa y el acceso directo en el escritorio
pues al estar el resto de documentos en la propia entrega se podrian producir duplicidades de archivos 
totalmente innecesarios si los incluia en el ejecutable de la instalacion.

Contenido:

Carpeta Java con el codigo java del proyecto
Carpeta Jar con el ejecutable jar si el profesor quiere ahorrarse la instalacion
Carpeta base de datos con todo lo relacionado para instalar la BD
Carpeta ejecutable con el instalador de windows
Documento del proyecto
Carpeta Otros documentos anexados con el resto de archivos necesarios
